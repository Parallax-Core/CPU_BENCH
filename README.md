Dependencies Used:
- Python 3.12
- PySide6
- psutil
- matplotlib
- numpy
- multiprocessing
- wmi
- and other standard librarires
- PyInstaller for compiling into a portablle executable without need of dependencies.

  
In order to Use this app, you can run the main_app.py on a shell or IDE by ("python main_app.py")
or run pyinstaller ("pyinstaller --onefile --windowed --add-data "style.qss:." --icon="cpu_benchmark.ico" main_app.py")
